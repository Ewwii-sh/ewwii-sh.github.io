precss

Predefined css classes for an easier time styling.

By: Byson94 Link: precss
CC0 style

precss is an ewwii plugin that provides a list of predefined classes to make it easier and faster to work with css. The classes that precss provides ranges from simple utility classes like rounded to animation classes like fade-in.
