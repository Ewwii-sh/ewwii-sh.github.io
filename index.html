Ewwii

“Building Widgets Made Better”

Ewwii is a lightweight, extensible, and easy to use widget system for building interactive widgets, status bars, and more on both Wayland and X11 quickly.

Get Started View on GitHub

Extensive Plugin Support

Use plugins to extend/replace core components of ewwii including the configuration language.

Hot Reload

Instantly see changes during development without having to restart your app for fast iteration.

Simple Configuration

Use a simple yet fast configuration language with an easy and intuitive syntax for building your widgets.

Ecosystem Tooling

An ecosystem of tools focused on extending core functionality and providing the best user experience.

Feather Weight

Extremely lightweight on both memory and binary size with the only overhead being GTK4 framework itself.

X11 + Wayland compatibility

Seamlessly create widgets on both X11 and Wayland environments with ease.
