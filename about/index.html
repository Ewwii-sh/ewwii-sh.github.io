About Ewwii

Ewwii is a lightweight, extensive, fully customizable GTK4 based widget system for building widgets, status bars, and more. It is a hard fork and a rewrite of eww. One of the core focuses of ewwii is simplicity, ease of use, good error diagnostics, and extensibility.

Features of ewwii:

  • Simple configuration language
  • Plugin system to extend core features
  • Excellent error diagnostics
  • Wayland and X11 support
  • GPU acceleration
  • Hot Reload

Origin of the Name

The name “ewwii” stands for Elkowars Wacky Widgets Improved Interface. It got this name because ewwii started out as a simple fork of eww (“Elkowars Wacky Widgets”), which did nothing but swap out the configuration language. Which we referred to as “interface”.

Is ewwii better than eww?

Ewwii started as a fork of eww but has since grown into a full rewrite. Compared to eww, Ewwii adds a plugin system, GPU acceleration, improved error messages, and fully dynamic widget creation and destruction at runtime. All of which eww does not support.

How to get started?

Getting started with ewwii is fairly simple. We use a simple configuration language that is designed to be easy. See the documentation to get started right away!
